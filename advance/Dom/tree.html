The CSSOM tree represents the parsed CSS rules. Each rule is converted into an object, organized by selectors and their respective styles. CSSOM Tree Representation

Hierarchy Visualization: CSSStyleSheet ├── Rule: body { background-color: #f2f2f2; font-family: Arial, sans-serif; } ├── Rule: h1 { color: blue; font-size: 36px; } ├── Rule: p { color: #666; font-size: 18px; } └── Rule: .container { width: 80%; margin: 40px auto; ... }

Render Tree The Render tree is a combination of DOM and CSSOM. It contains only the elements that will be visually displayed on the screen, with their computed styles. Render Tree Representation

Hierarchy Visualization: Render Tree ├── BODY (background-color: #f2f2f2; font-family: Arial, sans-serif) └── DIV (width: 80%; margin: 40px auto; ...) ├── H1 (color: blue; font-size: 36px): "Hello World" ├── P (color: #666; font-size: 18px): "This is a paragraph." └── BUTTON: "Click me!"
